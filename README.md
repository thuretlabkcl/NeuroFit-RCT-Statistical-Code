# NeuroFit Randomised Controlled Trial – Statistical Analysis Code

## Overview

This repository accompanies the manuscript:

**Farmand et al. 2026 - Effects of a three-month exercise programme on cognition, mood and neurogenesis: the NeuroFit randomised controlled trial**

It contains representative R Markdown scripts illustrating the statistical analyses performed in the study.

Representative examples are provided for each statistical modelling approach used throughout the manuscript. The same modelling framework was applied to the remaining outcomes by substituting the corresponding dependent variables and baseline covariates.

The corresponding datasets are available separately through the associated Figshare repository.

## Statistical methods

The repository includes representative implementations of:

- Beta generalised linear mixed-effects models
- Linear mixed-effects models
- Cumulative link mixed models
- Estimated marginal means
- Model diagnostics

---

## Software

Analyses were performed using R.

- glmmTMB
- nlme
- ordinal
- DHARMa
